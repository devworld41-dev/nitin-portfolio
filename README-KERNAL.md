# Nitin Portfolio (React + Vite)

## Run (dev)

Vite requires Node >= 20.19 (or >=22.12).

On this machine, start with Node 22 explicitly:

```bash
cd /Users/devworld/Desktop/nitin-portfolio
/Users/devworld/.nvm/versions/node/v22.22.0/bin/node node_modules/vite/bin/vite.js --host 0.0.0.0 --port 5173
```

If port 5173 is busy, it will auto-pick the next one.
