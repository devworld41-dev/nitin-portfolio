# Nitin Portfolio

Personal portfolio website for **Nitin Kishore**.

## Tech
- React + TypeScript
- Vite

## Local development

> Vite requires Node >= 20.19 (or >= 22.12).

```bash
pnpm install
pnpm dev
```

## Deploy

This repo is configured to deploy automatically to **GitHub Pages** via GitHub Actions on pushes to `main`.
